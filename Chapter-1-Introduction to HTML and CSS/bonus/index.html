HTML & CSS Quick-Reference Guide

HTML Tags

<div>

A generic container for other elements.

<div class="container">
    <p>This is inside a div container.</p>
</div>

<p>

Defines a paragraph.

<p>This is a paragraph.</p>

<img>

Inserts an image.

<img src="image.jpg" alt="Description">

<a>

Creates a hyperlink.

<a href="https://example.com">Link</a>

<ul>

Defines an unordered list.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>

Defines an ordered list.

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
</ol>

<form>

Creates an HTML form for user input.

<form action="/submit" method="post">
    <input type="text" name="name" placeholder="Enter your name">
    <input type="submit" value="Submit">
</form>

<input>

Defines an input control.

<input type="text" placeholder="Enter text">

CSS Properties

color

Sets the text color.

color: #333;

font-family

Sets the font family.

font-family: Arial, sans-serif;

background-color

Sets the background color.

background-color: #f0f0f0;

padding

Sets the padding around an element.

padding: 10px;

margin

Sets the margin around an element.

margin: 10px;

border

Sets the border around an element.

border: 1px solid #ccc;

text-align

Sets the alignment of text.

text-align: center;

font-size

Sets the size of the font.

font-size: 16px;
